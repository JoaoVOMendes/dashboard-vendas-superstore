# 📊 Dashboard de Vendas - Superstore

Projeto de análise de dados desenvolvido como parte do portfólio para a área de **Análise de Dados**.

---

## 🎯 Objetivo

Analisar o desempenho comercial de uma loja de varejo americana, identificando categorias mais lucrativas, sazonalidade nas vendas, eficiência logística e segmentos de clientes mais relevantes.

---

## 🛠️ Ferramentas Utilizadas

- **Excel** — limpeza, tratamento e enriquecimento dos dados
- **Power BI** — modelagem, medidas DAX e visualização

---

## 🔄 Etapas do Projeto

1. **Coleta** — Dataset público Superstore (Kaggle)
2. **Tratamento** — Tradução das colunas, correção de tipos, remoção de duplicatas
3. **Enriquecimento** — Criação de colunas calculadas: Margem %, Dias para Envio, Trimestre, Faixa de Venda, Rentável?
4. **Modelagem** — Criação de medidas DAX no Power BI
5. **Visualização** — Dashboard interativo com filtros por Ano, Trimestre e Categoria

## 💡 Insights Encontrados

### Vendas por Categoria
- **Furniture** lidera com maior volume de vendas
- **Technology** ocupa o segundo lugar
- **Office Supplies** tem o menor volume, mas boa rentabilidade

### Subcategorias mais lucrativas
- 🏆 Copiers, Phones e Accessories são as mais rentáveis

### Subcategorias no prejuízo
- ⚠️ Tables, Bookcases e Supplies operam com lucro negativo — indicando necessidade de revisão de precificação ou custos

### Segmentos de Clientes
- **Consumer** representa a maior fatia de vendas (53%)
- **Home Office** é o segmento com menor volume

### Regiões
- **West** lidera em vendas, seguida de **Central**
- **South** tem o menor desempenho regional
